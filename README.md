# KTV 包厢管理 - 双方案配色对比

## 文件说明

- **index.html** — ERP 桌面端，双方案切换（键盘 1/2 或方向键），右侧内嵌可拖拽小程序预览
- **minipro.html** — 小程序端，通过 postMessage 与 index.html 方案同步

---

## 配色关系

方案1（高饱和）是方案2（无渐变）的**同色系加深版**，色相一致，饱和度更高，对比度更强，文字更清晰。

---

## 两方案色值完整对照

| 状态 | 方案2 无渐变 | 方案1 高饱和 | 说明 |
|------|-------------|-------------|------|
| 空闲 | `#FFFFFF` ⬜ | `#FFFFFF` ⬜ | 白底，两方案相同 |
| 预定 | `#F0E6FF` 浅紫 | `#DFD0FF` 深紫 | 同色系加深 |
| 带客 | `#FFFFFF` ⬜ + 标签 `#B49EFF` | `#FFFFFF` ⬜ + 标签 `#9B85E3` | 白底 + 右上角紫色标签，标签色也加深 |
| 使用中 | `#C5E4F7` 浅蓝 | `#A0D8FF` 深蓝 | 同色系加深 |
| 临近超时 | `#FFE5AB` 浅黄 | `#FFD080` 深黄 | 同色系加深 |
| 已超时 | `#FFD4D4` 浅红 | `#FFB0B0` 深红 | 同色系加深 |
| 清洁中 | `#E8F5E8` 浅绿 | `#C8E8C8` 深绿 | 同色系加深 |
| 故障 | `#DFDEDD` 浅灰 | `#C8C5C3` 深灰 | 同色系加深 |

---

## 文字色规则（两方案通用）

| 元素 | 颜色 | 说明 |
|------|------|------|
| 房间名 | `#333` / `#1a1a1a` | 深色，方案1字重 700 更粗 |
| 时间 | `#333` / `#4b5563` | 深色 |
| 包厢类型 | `#444` / `#6b7280` | 次深色 |
| 价格（使用中） | `#e23939` 红色 | 突出显示 |
| 价格（临近超时） | `#c62828` 深红 | 警示 |
| 价格（已超时） | `#b71c1c` 更深红 | 强警示 |
| 带客标签文字 | `#FFFFFF` 白色 | 紫色背景上白字 |

## 小程序筛选按钮配色

| 状态 | 方案2 按钮色 | 方案1 按钮色 | 文字色 |
|------|-------------|-------------|--------|

---

## 功能说明

- 切换方案时 ERP 端和小程序端自动同步配色
- 小程序预览窗口默认右侧，可拖拽移动、可关闭/重新打开
- 键盘快捷键：`1` = 高饱和，`2` = 无渐变，方向键循环切换
