# KTV 包厢管理 - 双方案配色对比

## 文件说明

- **index.html** — ERP 桌面端，双方案切换（键盘 1/2 或方向键），右侧内嵌可拖拽小程序预览
- **minipro.html** — 小程序端，通过 postMessage 与 index.html 方案同步

---

## 方案1：高饱和（base-sat-w）

特点：彩色状态用高饱和背景 + 深色文字，非彩色状态统一白底

| 状态 | 背景色 | 色值 | 文字色 | 说明 |
|------|--------|------|--------|------|
| 空闲 | ⬜ 白色 | `#FFFFFF` | 深色 `#333` | 白底 + 灰色边框 |
| 预定 | ⬜ 白色 | `#FFFFFF` | 深色 `#333` | 与空闲同色，无特殊标记 |
| 带客 | 🟩 绿色 | `#6EE8A8` | 深色 `#1a1a1a` | 高饱和绿 |
| 使用中 | 🟦 蓝色 | `#80CFFF` | 深色 `#1a1a1a` | 高饱和蓝 |
| 临近超时 | 🟨 黄色 | `#FFD26E` | 深色 `#1a1a1a` | 高饱和黄 |
| 已超时 | 🟥 红色 | `#FF9797` | 深色 `#1a1a1a` | 高饱和红 |
| 清洁中 | ⬜ 白色 | `#FFFFFF` | 深色 `#333` | 白底 + 灰色边框 |
| 故障 | ⬛ 灰色 | `#C2BFBD` | 深色 `#1a1a1a` | 中性灰 |

## 方案2：无渐变（bottom-flat）

特点：淡色填充背景 + 深色文字，带客为白底 + 右上角紫色标签

| 状态 | 背景色 | 色值 | 文字色 | 说明 |
|------|--------|------|--------|------|
| 空闲 | ⬜ 白色 | `#FFFFFF` | 深色 `#333` | 白底 + 灰色边框 |
| 预定 | 🟪 浅紫 | `#F0E6FF` | 深色 `#333` | 淡紫色填充 |
| 带客 | ⬜ 白色 | `#FFFFFF` | 深色 `#333` | 白底 + 右上角紫色标签 `#B49EFF` |
| 使用中 | 🟦 浅蓝 | `#C5E4F7` | 深色 `#333` | 淡蓝色填充，价格红色 `#e23939` |
| 临近超时 | 🟨 浅黄 | `#FFE5AB` | 深色 `#333` | 淡黄色填充，价格红色 `#c62828` |
| 已超时 | 🟥 浅红 | `#FFD4D4` | 深色 `#333` | 淡红色填充，价格红色 `#b71c1c` |
| 清洁中 | 🟩 浅绿 | `#E8F5E8` | 深色 `#333` | 淡绿色填充 |
| 故障 | ⬛ 浅灰 | `#DFDEDD` | 深色 `#333` | 浅灰色填充 |

## 功能说明

- 小程序预览窗口默认在右侧，可拖拽移动、可关闭/重新打开
- 切换方案时 ERP 端和小程序端自动同步配色
- 键盘快捷键：`1` = 高饱和，`2` = 无渐变，方向键循环切换
